# processkiller

Kills linux processes based on a name pattern.

## Usage examples

```
./processkiller.sh sleep
./processkiller.sh -i Firefox
```

#### Output

```
$ ./processkiller.sh -i Sublime
Sublime [373680] killed
Sublime [373695] killed
```
